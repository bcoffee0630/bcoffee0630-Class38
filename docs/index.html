龍華科技大學 · 動漫遊戲設計

龍華科大自1999年起成立遊戲系，屬全臺第一所將「遊戲」二字示名於系名的科系；
為了讓海外學子亦能夠涉足多媒體領域而設立海青專班動漫與遊戲設計製作科，
招收來自馬來西亞、印尼等地之學生，旨在培養出更多兼具理論與實務的遊戲暨設計人才。
